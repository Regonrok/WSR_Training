Мобильное приложение Wsr_forms_
==================

Наше приложение позволяет работать с Desctop версией WSR_FORMS_, Но на вашем телефоне под Android
------------------------------------
### Меню в приложение

#### Главная

 На главной форме расположены данные для ввода: 
	
	1. Имя
	2. Фамилия
	3. Отчество

 После находятся формы для выбора:
	
	1. выбор Темперамента
	2. выбор Характера
	3. выбор Знака зодиака
	4. выбор Вида спорта

Так же ниже находится форма выбора аватарки, при выборе фото меняется сама аватарка

#### Зодиаки

При выборе знака зодиака меняется основной знак зодиака

#### Темперамент

На данной форме распаложен тест. Выбор варианта теста зависит ваш темперамент

#### Спорт

На данной форме можно выбрать до 2 видов спорта которым вы увлекаетесь.
Или нажатием на кнопку Сброс убрать все виды спорта.
